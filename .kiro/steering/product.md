# Pandora — Product Overview

Pandora is a desktop developer toolbox built with Tauri 2, combining four core modules into a single IntelliJ-style draggable panel interface (powered by dockview):

1. **API Tester** — HTTP request builder with collections, environments, variables, history, and cookie management
2. **DB Manager** — Multi-database client supporting SQLite, MySQL, and PostgreSQL with query editor (Monaco), schema browsing, favorites, and batch import
3. **Toolkit** — Extensible utility tools (encoding, crypto, network, text) with a plugin system that runs utools-compatible plugins in iframe sandboxes
4. **Script Runner** — Multi-runtime script execution (Node.js, Python, etc.) with a built-in editor, file management, and streaming output

The app targets Windows, ships as a Tauri bundle, and uses Chinese as the default UI language with English as a secondary locale.

App identifier: `com.pandora.app`
